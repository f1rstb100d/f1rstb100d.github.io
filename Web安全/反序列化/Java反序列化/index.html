    Java反序列化¶

    什么是Java序列化¶

    1. Java 序列化是指把 Java 对象转换为字节序列的过程。
    2. Java 反序列化是指把字节序列恢复为 Java 对象的过程。

    实现¶

    使用到JDK中关键类 ObjectOutputStream(对象输出流) 和**ObjectInputStream**(对象输入流)

    ObjectOutputStream 类中：通过使用 writeObject(Object object) 方法，将对象以二进制格式进行写入。

    ObjectInputStream 类中：通过使用 readObject()方法，从输入流中读取二进制流，转换成对象。

    import java.io.*;

public class main {
     main() {
    }

//序列化函数
    public static byte[] serialize(Object obj) throws IOException {
        ByteArrayOutputStream baos = new ByteArrayOutputStream();
        ObjectOutputStream oos = new ObjectOutputStream(baos);
        oos.writeObject(obj);     
        return baos.toByteArray();
    }

//反序列化函数
    public static Object unserialize(byte[] bytes) throws IOException, ClassNotFoundException {
        var ois = new ObjectInputStream(new ByteArrayInputStream(bytes));
        return ois.readObject();
    }
}

    注：只有实现Serializable接口的类才可以进行反序列化与序列化
